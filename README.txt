Bienvenue dans l'antre du Groupe 19.

Pour compiler notre programme, un simple " make " dans le terminal suffit.
Pour lancer le jeu, il suffit de taper " ./main ", mais veillez à le lancer
avec le terminal en plein écran, ou au moins aux deux tiers.

Les options de lancement sont les suivantes :
--help : affiche l'aide
--version : affiche la version
--auteurs : affiche les auteurs

Vous pouvez toutes les utiliser en même temps, et si vous ne tapez pas une bonne
option, pas de soucis, un message s'affiche vous répetant les options possibles.

Aussi, merci de NE PAS MODIFER LE FICHIER CLASSEMENT.TXT, sous peine de bugs.
L'unique condition pour pouvoir le faire est d'entrer des chiffres inférieurs à
99 999 et des noms à 3 caractères non-spéciaux, sans changer les séparateurs.
